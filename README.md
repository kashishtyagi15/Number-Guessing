# 🎯 Java Swing Number Guessing Game

A simple Number Guessing Game developed using Java Swing. Guess the randomly generated number within limited attempts, earn points based on your performance, and play multiple rounds.

## ✨ Features
- 🎲 Random number generation (1–100)
- 🎯 5 attempts per round
- 💡 Too High / Too Low hints
- 🏆 Score tracking
- 🔄 Multiple rounds support
- ✅ Input validation
- ❌ Cancel game anytime

## 🛠️ Technologies
- Java
- Java Swing
- JOptionPane
- Random Class
- OOP

## 🎮 How to Play
- Enter a number between **1 and 100**.
- You have **5 attempts** to guess the correct number.
- Receive hints after every incorrect guess:
  - ⬆️ Too High
  - ⬇️ Too Low
- Earn more points by guessing the number in fewer attempts.
- Choose **Yes** to play another round or **No** to end the game.

## ▶️ Run the Project

```bash
javac NumberGuessingGame.java
java NumberGuessingGame
```
